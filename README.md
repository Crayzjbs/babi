# Love Letter Website

An immersive, scroll-driven love letter website with cinematic animations and editorial design aesthetic.

## Features

- **Smooth Scrolling**: Lenis for ultra-smooth inertial scrolling
- **Scroll Animations**: GSAP + ScrollTrigger for parallax and entrance effects
- **Typewriter Effect**: Sequential character-by-character text reveal
- **Ambient Music**: Background music with mute control
- **Floating Elements**: Parallax photos and GIFs at different depths
- **Grain Texture**: Analog film photography aesthetic
- **Particle Background**: Subtle floating particles
- **Mobile Responsive**: Optimized for iPhone 14 and all devices

## Technologies

- GSAP + ScrollTrigger
- Lenis (smooth scrolling)
- Splitting.js (character animation)
- Anime.js (typewriter effects)
- tsParticles (ambient background)
- Three.js

## Deployment

This site is deployed on Vercel and can be accessed via QR code.

## Local Development

Simply open `index.html` in a modern web browser.
